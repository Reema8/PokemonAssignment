# How to run this application
1. Run 'npm intall'
2. npm start to see the website.

# Description of application
1. Dashboard: Details of all Pokemons
2. Details Button: Detail of particular Pokemon
3. Search Textbox: Search any Pokemon and then press enter. Will show required Pokemon
4. Next and Previous Buttons for Pagignation
